✨ Features at a Glance

  • ⚡️ Instant Sessions
    Start and stop with zevfocus start/stop.
  • 🛡 Tamper‑Proof
    HMAC signatures + hash to prove authenticity.
  • 🔥 Daily Streaks
    Dopamine‑driven streak counter like GitHub.
  • ⏱ Idle Detection
    Counts only real keyboard/mouse activity.
  • 📊 Beautiful Summaries
    CLI dashboards with emojis & structure.
  • 🐦 One‑Click Sharing
    Generate tweet‑ready cards + hash IDs.

🚀 Quick‑Start

$ zevfocus login --user "YourName"
$ zevfocus start --taskname "Write blog post"
# ...focus...
$ zevfocus stop
$ zevfocus summary
$ zevfocus share   # generates tweet + hash
